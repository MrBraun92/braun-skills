# Requisitos Técnicos por Cargo AI (2025-2026)

## AI/ML Engineer
Must-have: Python, PyTorch ou TensorFlow, Git, Docker
Nice-to-have: Kubernetes, MLflow, Weights & Biases, AWS/GCP/Azure
Diferencial: LLM fine-tuning, RLHF, distributed training

## MLOps Engineer
Must-have: Python, Docker, Kubernetes, CI/CD, cloud (AWS/GCP/Azure)
Nice-to-have: Kubeflow, Airflow, MLflow, DVC, Feast
Diferencial: Feature stores, model monitoring, A/B testing infra

## LLM Engineer
Must-have: Python, LangChain ou LlamaIndex, OpenAI/Anthropic APIs, RAG
Nice-to-have: Vector DBs (Pinecone, Weaviate, Chroma), fine-tuning, RLHF
Diferencial: Multi-agent systems, custom embeddings, production LLM optimization

## Data Scientist
Must-have: Python, SQL, estatística, scikit-learn, pandas
Nice-to-have: R, Spark, dbt, Tableau/Looker, A/B testing
Diferencial: Causal inference, time series, NLP

## AI Product Manager
Must-have: Experiência com produtos tech, entendimento de ML básico
Nice-to-have: SQL, Python básico, experiência com dados
Diferencial: Experiência com produtos AI em produção, métricas de modelo

## Certificações Valorizadas
- AWS Certified Machine Learning Specialty
- Google Professional Machine Learning Engineer
- Deep Learning Specialization (Coursera/DeepLearning.AI)
- Hugging Face NLP Course
