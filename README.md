# Manus Skills Repository

This repository contains all installed **Manus Agent Skills** — modular capability extensions that enhance the agent's behavior in specialized domains.

Each skill lives in its own directory and contains a `SKILL.md` file with instructions, scope, and usage guidelines.

---

## How to Use

- **Edit a skill**: Navigate to the skill's folder and edit its `SKILL.md` file directly on GitHub.
- **Re-import to Manus**: Share the raw URL of a `SKILL.md` file or the repository link with Manus or Claude.
- **Add a new skill**: Create a new folder with the skill name and add a `SKILL.md` inside it.
